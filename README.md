# Peace
Peace by Dennelle is based on the classic War Game with a positive twist.

### Technologies:
☐ HTML
☐ CSS
☐ Javascript

### Getting Started:
Peace is a two-player game against the computer.

To get started click the face down player 1 card. Click the card on every play.

### Objective:
The object of the game is to keep your peace by drawing a higher card than the 2nd player. If you match ranks with the other player you have entered a War scenario. In a War scenario both players lose 4 points, not cards. The player with the most points after all the cards are played wins. Peace is more efficient than the original War game and it helps people to be more aware of their vibration.

### Features:
☐ Feature 1: Cards are automatically shuffled

☐ Feature 2: Scores are added and subtracted

☐ Feature 3: Plays music on click

☐ Feature 4: Screen changes when War is on.

☐ Feature 5: CSS keyframe


### Next Steps:
The next iteration of Peace includes:

-Replacing cards with gemstones that have power levels. If your gemstones matches another's players gemstone you have to answer a trivia question in order win all the points.

-Adding a 3 or 4 second timer between card play.
